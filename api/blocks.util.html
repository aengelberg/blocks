check

macro

(check value pred? & on-err)

Utility macro for validating values in a threading fashion. The predicate pred? will be called with the current value; if the result is truthy, the value is returned. Otherwise, any forms passed in the on-err list are executed with the symbol value bound to the value, and the function returns nil.
